Jotai
===================

Jotai is an open source hardware monitoring library.

(Jotai was originally an http://openhardwaremonitor.org fork)

----------
## System support
> - Windows (x86, x64)
> - Linux (x86, x64)

## Dependencies
> - Newtonsoft.Json >= 7.*
> - Windows: .NET Framework 4
> - Linux: Mono 4.0
